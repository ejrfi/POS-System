## Packages
recharts | Dashboard charts for sales trends and analytics
framer-motion | Smooth animations for cart items, modals, and page transitions
lucide-react | Beautiful icons for the UI
zustand | Client-side state management for the shopping cart

## Notes
- The POS system uses a Sidebar layout.
- Authentication is simulated or handled via `useAuth` hook pattern connecting to /api/user.
- Barcode scanning relies on focused input (standard USB scanner behavior).
- "Check out" creates a sale transaction.
